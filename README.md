# 💰 Finance Tracker UI

A mobile app UI design built with Figma for tracking personal finances.

## 📱 Screens
| Screen | Description |
|---|---|
| 01 - Onboarding | Welcome screen with CTA |
| 02 - Dashboard | Balance overview & transactions |
| 03 - Add Transaction | Form to log income/expenses |
| 04 - Analytics | Bar chart & spending breakdown |
| 05 - Profile | User settings & account info |

## 🎨 Design
- **Tool:** Figma (Free)
- **Style:** Dark theme
- **Colors:** #0D0D0D · #4F9EFF · #1A1A2E
- **Font:** Inter
